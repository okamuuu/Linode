screen導入

仮想端末ソフトであるscreenは
1つの端末の中に複数のウィンドウを立ち上げて作業することができます。

確認

screenが導入されているかどうかを確認します。 

$ sudo aptitude search screen
p   screen - terminal multiplexor with VT100/ANSI terminal emulation

インストール 

$ sudo aptitude update
$ sudo aptitude install screen

設定ファイルを編集

次のように編集します。

  • screenのエスケープキャラクタをCtrl+tにする
  • ビジュアルベルをオフにする
  • 画面の最下部にウィンドウ番号などを表示する。
  • 自動デタッチ機能を有効にする。
  • 文字コードにUTF8を使う。
  • エスケープキャラクタ＋kでウィンドウを閉じるようにする

※私はCtrl+tがエスケープキャラなのですが、Windowsマシン上でこの とおりに指を広げるのは案外つらいのでCapsLockとCtrlの位置を入れ替えて使用しています。 

$ vi ~/.screenrc

# common
escape ^Tt
vbell off
autodetach on
startup_message off
defscrollback 10000

# status line
hardstatus ignore "%w | %h"
hardstatus alwayslastline " %`%-w%{=b bw}%n %t%{-}%+w"
caption always "%{= wk} %-w%{=bu dr}%n %t%{-}%+w %= %{=b wb}%y/%m/%d %{=b wb}%c"

# utf8
defencoding utf-8
encoding utf-8 utf-8
