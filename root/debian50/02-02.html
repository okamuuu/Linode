サービス確認

現在動作中のサービスを確認します

service導入

RedHat系ユーザーにとっては驚く事かもしれませんがdebianにはchkconfigとserviceがはいっていません。ですのでこれを最初にインストールします。 

$ sudo aptitude update
$ sudo aptitude install chkconfig

/sbin/chkconfigを使って、サーバーが起動したら自動的に立ち上がるプロセスを確認してみます。
