subversion

ソースコードのバージョン管理を行います。

インストール

% sudo yum install subversion

subversionリポジトリを準備

% sudo mkdir -p /var/svn
% cd /var/svn

% sudo svnadmin create /var/svn/Root
% sudo chown -R apache:apache /var/svn/Root

% ls /var/svn/Root
README.txt  conf  dav  db  format  hooks  locks

http経由でアクセスする

VirtualHostの設定 

<VirtualHost *:80>
    ServerName svn.mydomain.com
    ServerAdmin webmaster@mydomain
    HostnameLookups Off

    <Location />

       Order deny,allow
       Deny from all
       
       ### アクセス制限
#       Allow from 127.0.0.1
#       Allow from 192.168.1.0/24 

       DAV svn
       SVNPath /var/svn/Root

       # Limit write permission to list of valid users.
       <LimitExcept GET PROPFIND OPTIONS REPORT>
           AuthType Digest
           AuthName svn_users
           AuthUserFile /var/svn/users.htdigest
           Require valid-user
       </LimitExcept>

    </Location>
</VirtualHost>

変更された設定を反映させる 

% sudo /sbin/service httpd restart

htdigest認証用ユーザー登録

subversion用のユーザーを作成 % sudo htdigest -c /var/svn/users.htdigest svn_users svnuser
