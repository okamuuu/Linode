LinodeのDNSサーバーを利用

WEBサーバーを公開するに際に必要なものはDNSサーバです。
Linodeでは無料で利用できるDNSサーバーがあるのでこれを利用します。

レジストラへLinodeのDNSサーバーを登録する。

私の場合、ドメインをムームードメインで取得しました。 そうするとレジストラはムームードメインとなりますので、ムームードメインの管理画面でDNS登録のサービスを行います。

ムームードメインでwww.example.jpというドメインを取得していた場合は、 その他ネームサーバーのネームサーバという項目を選択してネームサーバー次のように登録します。

ネームサーバ１ ns1.linode.com
ネームサーバ２ ns2.linode.com
ネームサーバ３ ns3.linode.com
ネームサーバ４ ns4.linode.com

DNS Manager

Linode側ではメンバーページからDNS Managerへ遷移して下さい。 Add a new domain zoneをクリックして必要な項目を入力します。
