= cpan packager =

'''''CPANモジュールの依存関係を自動的に解決してDebパッケージを作成します。'''''

== cpan packagerとは ==

CPANモジュールの依存関係を''再帰的''に解決して自動的にRPM/Debパッケージを作成する開発ツールです。''cpan''コマンドを使っても良いのですが、サーバー毎に違うVersionのモジュールがインストールされてしまう事により発生する手違いを防ぎたい場合があります。

既存ツールにも似たものがありますが、これは依存関係を解決する、という特徴があります。

== CPAN::Packagerを導入 ==

{{{
% sudo su
# cpanm CPAN::Packager
}}}
